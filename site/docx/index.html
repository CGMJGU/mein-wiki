    Das Dateiformat .docx

    Was ist eine .docx-Datei?

    Eine .docx-Datei ist ein Textdokument, das mit Microsoft Word erstellt wird. Das Format speichert nicht nur den Text, sondern auch die gesamte Formatierung – also Schriftarten, Abstände, Tabellen, Bilder und Seitenumbrüche.

    Das „x" am Ende steht für das XML-Format, das seit Word 2007 verwendet wird.

    Womit kann man .docx öffnen?

    Programm Betriebssystem Kostenlos?
    Microsoft Word Windows, macOS Nein (Lizenz nötig)
    LibreOffice Writer Windows, macOS, Linux Ja
    Google Docs Browser (alle Geräte) Ja
    Pages macOS, iOS Ja

    !!! warning "Achtung" Auch wenn ein Programm .docx öffnen kann, sieht das Dokument nicht immer gleich aus. Schriftarten, Abstände oder Tabellen können sich verschieben – je nachdem welches Programm verwendet wird.

    Proprietär oder offen?

    .docx ist ein proprietäres Format – es gehört Microsoft. Das bedeutet:

    • Es wurde von Microsoft entwickelt und wird von Microsoft kontrolliert
    • Andere Programme können es zwar lesen, aber manchmal nicht vollständig korrekt darstellen
    • Auf manchen Geräten (z. B. älteren Smartphones) lässt es sich gar nicht öffnen

    !!! info "Zum Vergleich" Ein offenes Format wie .pdf oder .txt ist öffentlich dokumentiert – jeder darf Software dafür schreiben, und es funktioniert überall gleich.

    Wann ist .docx sinnvoll?

    .docx eignet sich dann, wenn das Dokument noch bearbeitet werden soll – zum Beispiel:

    • Jemand anderes soll den Text noch ändern oder ergänzen
    • Du arbeitest gemeinsam mit jemandem an einem Dokument
    • Du schickst einen Entwurf zur Korrektur

    Wann ist .docx nicht sinnvoll?

    • Wenn das Dokument fertig ist und nur gelesen werden soll
    • Wenn du nicht weißt, welches Programm die Empfängerin hat
    • Wenn das Layout auf jedem Gerät gleich aussehen soll
    • Wenn die Datei auf einem Smartphone geöffnet werden soll

    Dateigröße

    .docx-Dateien sind meist klein – ein normaler Textbericht hat nur wenige Kilobyte. Sobald Bilder eingebettet werden, kann die Dateigröße aber stark ansteigen.
