## Was sind Cloud-Dienste?

Cloud-Dienste wie **OneDrive** (Microsoft) oder **Google Drive** speichern Dateien auf einem Server im Internet. Statt eine Datei als Anhang zu schicken, teilt man einen **Link** zu einem Ordner auf welchen man über das Internet zugriff hat. 

Der Zugriff ist von überall möglich, das Größenlimit ist großzügig, und wer seine Dateien mit der Cloud synchronisiert hat immer automatisch die aktuellste Version verfügbar.

Allerdings liegen die Daten auf fremden Servern, was ein **Datenschutzrisiko** darstellen kann. Außerdem benötigt der Sender in der Regel einen Account und eine Internetverbindung.

Cloud-Dienste eignen sich besonders wenn man mit anderen zusammenarbeiten oder Dateien mit mehreren Personen teilen möchte.
